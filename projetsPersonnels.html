Afficher une caméra Windows 11 sur une VM Windows 95

Technos : C, DosBox, WSL

Contexte : Réalisation d'une VM DOSBox contenant Windows 95 étant relié à Windows 11 pour transmettre un flux vidéo

Ce que j’ai fait : Le But était de faire comprendre à un dinosaure comment afficher une vidéo dans un langage compréhensible. J'ai donc codé un Serveur en C qui envoie des données retravaillées à un client sur un pont Réseau via Socket TCP.

Bilan : Cela m'a fait comprendre les limites des OS, les outils et comment les systemes communiquent et évoluent

Simulation d'OS & Environnement de Bureau

Technos : Python (POO), Bibliothèques graphiques natives

Contexte : Défi technique personnel : recréer l'expérience utilisateur de Windows 95/98 entièrement en code moderne

Ce que j’ai fait : Architecture complète d'un "Système d'Exploitation" simulé :
• Système de Fichiers : Développement d'une couche d'abstraction pour naviguer sur la machine hôte via l'interface rétro.
• Applications Natives : Recodage complet d'outils fonctionnels (Bloc-notes, Explorateur de fichiers, Terminal,...).

Bilan : Une plongée profonde dans la programmation événementielle et la structure des systèmes graphiques.

Réseau de Neurones

Technos : Java

Contexte : Création d'une Intelligence Artificielle capable d'apprendre à survivre seule dans un environnement sans aucune règle pré-codée

Ce que j’ai fait : Développement d'un moteur d'apprentissage par renforcement "from scratch" :
• Réseau de Neurones : Codage manuel des perceptrons, couches cachées et fonctions d'activations.
• Sélection Naturelle : Implémentation d'un algorithme génétique (mutation, croisement) pour faire évoluer les générations.

Bilan : Une démonstration visuelle du Deep Learning : passage de créatures aléatoires à des survivants en 300 générations.
